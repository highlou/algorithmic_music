# algorithmic_music
Creates music algorithmically!

I might make `synth.hh` and `synth.cc` their own repository later on.

## Build

### Dependencies

* GCC Toolchain, including `make` and `g++`
* SDL2, both the shared object and development files, should be able to be
linked by using `-lSDL2`

### Actually building it

```sh
make build
```
